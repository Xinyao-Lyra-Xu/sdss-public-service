# SDSS Datathon 2026 — Public Services (Shelter Occupancy & Capacity)

## Project Goal
Assess system pressure, capacity utilization, and operational risk over time, and propose data-driven recommendations.

## Repo Structure
- `data/`: dataset (not tracked in git)
- `notebooks/`: analysis notebooks (submit fully executed .ipynb)
- `src/`: helper functions (optional)
- `reports/`: exported figures/tables
- `slides/`: final slides (PDF)
